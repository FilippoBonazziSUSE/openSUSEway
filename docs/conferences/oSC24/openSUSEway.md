## Sway

- openSUSE theme, wallpaper
- touchpad
- lock screen (swaylock)
- media keys (volume, media, brightness)
- screenshot menu (grim)
- system mode menu
- audio / brightness slider (wob)
- Polkit authentication agent (GNOME)
- Floating windows

- Window title
- Tray area
- zypper widget
